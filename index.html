Introducción a los Sistemas Operativos

¿Qué es un Sistema Operativo?

Un sistema operativo es el software principal que gestiona el hardware del computador y proporciona servicios para las aplicaciones. Actúa como intermediario entre el usuario y el hardware.

Funciones Principales

  • Gestión de procesos y recursos
  • Administración de memoria
  • Control de dispositivos de entrada/salida
  • Gestión del sistema de archivos
  • Interfaz de usuario

Tipos de Sistemas Operativos

Windows

Sistema operativo de Microsoft, ampliamente usado en entornos empresariales y personales.

Linux

Sistema operativo de código abierto, popular en servidores y desarrollo.

macOS

Sistema operativo de Apple, conocido por su interfaz intuitiva y estabilidad.

Gestión de Procesos

Concepto de Proceso

Un proceso es un programa en ejecución que incluye el código, datos, y el estado actual de ejecución.

Estados de un Proceso

  • Nuevo: El proceso se está creando
  • Listo: Esperando ser asignado al procesador
  • Ejecución: Las instrucciones se están ejecutando
  • Espera: Esperando por algún evento
  • Terminado: El proceso ha finalizado

Planificación de Procesos

Los algoritmos de planificación determinan qué proceso se ejecuta en un momento dado:

  • FCFS (First-Come, First-Served)
  • Round Robin
  • Prioridades
  • Shortest Job First

Gestión de Memoria

Jerarquía de Memoria

El sistema de memoria está organizado en una jerarquía que balancea velocidad, capacidad y costo.

Técnicas de Gestión

  • Memoria Real: Acceso directo a la memoria física
  • Memoria Virtual: Extensión de la memoria usando almacenamiento secundario
  • Paginación: División de memoria en páginas de tamaño fijo
  • Segmentación: División lógica basada en unidades de programa

Problemas Comunes

Fragmentación

Espacios de memoria no utilizados que no se pueden asignar

Intercambio

Mover procesos entre memoria principal y secundaria

Paginación por Demanda

Cargar páginas solo cuando son necesarias

Gestión de Almacenamiento

Sistemas de Archivos

Organización y estructuración de datos en dispositivos de almacenamiento.

Tipos de Sistemas de Archivos

  • FAT32: Sistema antiguo, compatible universalmente
  • NTFS: Sistema moderno de Windows con características avanzadas
  • ext4: Sistema de archivos común en Linux
  • APFS: Sistema de archivos de Apple

Operaciones de E/S

Gestión de dispositivos de entrada/salida:

  • Controladores de dispositivos
  • Buffering y caching
  • Planificación de discos
  • RAID y redundancia
